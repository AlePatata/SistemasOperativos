# SistemasOperativos

Curso desarrollados durante el semestre de primavera 2024, del profesor Luis Mateu, dictado por el profesor Lucas Torrealba.
- T1 sobre paralelización
- T2 sobre sincronización con mutex y una sola condición
- T3 sobre sincronización con control de acceso con múltiples condiciones
- T4 sobre programación de herramientas de sincronización sin timeout
- T5 sobre programación de herramientas de sincronización con timeout
- T6 sobre núcleos modernos y sincronización con spin-locks
- T7 sobre programación de módulos de Linux
